# WhatsApp Auto Messenger

Send personalized WhatsApp messages automatically from your computer using Python and Selenium — with support for image attachments and scheduling!

> Developed by [THATHEU CYRIL J], this script helps you automate routine WhatsApp communication, ideal for small businesses, campaigners, customer engagement, or personal reminders.

---

## 🚀 Features

- Send messages to any number
- Attach and send images with message
- Schedule delivery with delay
- Automatically skip if WhatsApp is not installed or number inactive
- Works on standard WhatsApp Web

---

## ⚙️ Requirements

- Python 3.8+
- Google Chrome
- ChromeDriver (match your Chrome version)
- Selenium

pip install selenium
